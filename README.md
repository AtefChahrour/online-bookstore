Online Bookstore 

A simple online bookstore built with React,And Vite
Users can browse books, add items to the cart, and perform an order.
Pages:
- Home  
- Books  
- About  
- Contact  
- Cart

 Books Page  
- Displays books  
- Clicking a book adds it to the cart

Technologies Used:

- React
- Vite
- React Router DOM
- CSS
